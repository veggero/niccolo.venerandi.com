Nylo

Types of structures:

Structure:

(int x, int y)

Dictionary (structure with defined
arguments):

(int x: 3, int y: 4)

Function (structure with
undefinible arguments):

(int x, int y: x + 3)

How do structures react:

Two structures reacting:

(int x, int y)(int z) = (int x, int y, int z)

Structure and dictionary:

(int x, int y)(int x: 3, int z: 4) = \
(int x: 3, int y, int z: 4)

Function and dictionary:

(int x, int y: x + 3)(int x: 4) = \
(int x: 4, int y: 7)

Manipulating structures

Setting new values or calling:

(int x: 3, int y: x+1)(int x: 10) =\
(int x: 10, int y: 11)

Saving structure value to a local variable:

(int x: 3)(x -> something)

Getting a value from a structure:

1 + (int x: 3)(<- x) = 4

A mix of them:

(int y: x + 3, int z: 0)(int x: 4, y -> height, <-z)

Types

int a: 3,
str b: "hello",
float c: 4.12,
struct d: (int x),
fun e: (int x, int y: x + 1),
dict f: (a: 1, b: 2),

list int g: [1, 2, 3],
dict int h: (a: 1, b: 2)

Functions

How to declare a function:

sum: (int a, int b, int result: a + b)

How to call a function:

sum(a: 3, b: 4, <-result)

A bit more complex function:

range: (
    int from,
    int to,
    list int numbers: from..to,
    int length: to-from
)

Assigning function values to variables:

range(
    from: 0,
    to: 100,
    numbers -> to_check,
    length -> range_len
)

Classes

A simple class:

point: (
    int x,
    int y,
    str repr: "Coord {x} {y}"[x: x, y: y, <-formatted]
)

Creating an instance:

point c: point(x: 2, y: 10)

Operating on classes:

move(
    int delta_x,
    int delta_y,
    point to_move,
    
    int old_x: to_move(<- x),
    int old_y: to_move(<- y)
    
    point moved_point: point(x: old_x+delta_x, y: old_y+delta_y)
)

What methods should be in classes, and what globals?
Classes methods are the one that only use class
data without any other parameter. Class method should
never change values.
Examples: to_str, repr, length, sorted, formatted.
Data should only be modified by external functions.

Making things a little easier

Calling a function with values, without specifing
where should they go, will set them to a variable of
the same type:

(int a, str b, bool c)("hey", 24, true) =\
(int a: 24, str b: "hey", bool c: true)

If there are two empity variables of
the same type, and two values of that type
are given, the output will be random:

(int x, int y)(1, 2) = (int x: 2, int y: 1)

You can leave structure variables unnamed, and rather use
their class name:

(int, result: int + 3)
(int, str, result: str * int)
(int, int, result: int[0] + int[1])

Also, if a structure has the value _self, it will be
always returned:

(int _self: 4) = 4
(int x, int _self: x+10)(int x: 3) = 13

So, this is a simple sum:

(int, _self: int+1)(3) = 4

Also, you can semplify proprieties getting:

(x: 3)(<-x) = (x: 3).x

Blocks

You can make multiline code using {}:

main: {
    print("a")
    print("b")
}

Blocks are allowed also in structures:

point: (
    int x, int y,
    
    // This is ugly af. Just take it
    // as example.
    int write_down: {
        print(x)
        print(y)
    }
)

Meta Blocks

Meta Blocks also has information
about the following code.
Syntax is {structure | code}

{int element | print("Hello")}

In Meta Blocks, the code will become
_self of the structure.

{int x | x + 1} = (int x, _self: {x + 1})

There are two ways of using Meta Blocks.
First, it's useful to create functions:

{int x | x + 1}(x: 4) = 5

Also, it can be used with iter, if, while, etc:

if {2<3 | print("yay")}

Flow Control

if and while:

if {2>3 | print("yay")}
while {2<3 | print("yay")}

iterating on lists and repeating:

iter {[1, 2, 3], n | print(n)}
repeat {3 | print("yay")}

repeat and iter will set _self to
the value of their code:

iter {[1, 2, 3], n | n + 1} =\
[2, 3, 4]
repeat {5 | 3} = [3, 3, 3, 3, 3]

Curried Functions

You don't have to provide all the
arguments to a function with the
first call.
Let's take a general function
to sum two integers:

sum: {int, int | int[0] + int[0]}
sum(1, 2) = 3

What happens if we provide only
one integer?

sum_to_two: sum(2)
sum_to_two(7) = 9
sum_to_two(4) = 6

This works also with args/classes:

point: (int x, int y)
x_axis: point(y: 0)
y_axis: point(x: 0)
c: y_axis(y: 5)

Conditions

You can also add conditions to
types:

(int[<3] x) (0) // okay
(int[<3] x) (4) // error

You can add conditions to
other elements:

(list [list.length > 5]
int [int % 3 = 0]
x)
